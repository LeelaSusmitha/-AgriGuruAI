# 🌾 AgriGuru AI

An AI-powered Smart Farming Assistant that helps farmers with crop disease detection, weather advisory, multilingual support, and AI-based agricultural guidance.

---

## 🚀 Features

### 🤖 AI Farming Assistant
- Ask agriculture-related questions
- AI-generated farming guidance
- Multilingual support

### 🌾 Crop Disease Detection
- Upload crop images
- AI-powered disease diagnosis
- Treatment recommendations
- Prevention tips

### 🌦 Weather Advisory
- Real-time weather information
- Temperature monitoring
- Humidity analysis
- Weather-based farming suggestions

### 🌍 Multi-Language Support
- English
- Telugu
- Hindi

### 📊 Smart Dashboard
- AI interaction tracking
- Crop analysis statistics
- Image upload analytics
- User activity overview

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/LeelaSusmitha/-AgriGuruAI.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside backend folder.

```env
GROQ_API_KEY=YOUR_API_KEY
OPENWEATHER_API_KEY=YOUR_API_KEY
```
